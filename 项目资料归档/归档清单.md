# NSTI 项目资料归档清单

本文件夹用于集中存放非网页运行所必需的文本、说明、临时数据、提交材料与辅助脚本。归档后，项目根目录主要保留网站实际运行需要的文件，便于后续部署和维护。

## 根目录保留内容

- `.git/`
- `.gitignore`
- `.vercelignore`
- `index.html`
- `app/`
- `api/`
- `项目资料归档/`

## 说明

网站页面与接口仍保留在根目录的 `index.html`、`app/`、`api/` 中。`项目资料归档/` 已加入 `.vercelignore`，后续部署时不会把这些说明材料和临时资料一起上传。
