# MBTI / “STi 测试”在说什么（简要背景）

## 你在网上看到的“STi”通常指什么

- 互联网上流行的“STi/sti测试”多数是对“性格类型测试”的泛称，最常见指向是 MBTI 及其网络化版本（例如 16Personalities）。
- MBTI 的叙事来自荣格类型学：用若干二分“偏好维度”组合出 16 种类型（例如 INTP、ESFJ）。
- 16Personalities 结果展示沿用“四字母类型”，但题目与计分更接近“大五人格”风格，并额外加入 A/T 维度，因此不等同于经典 MBTI。

## 常见维度（为什么总是看到那四个字母）

MBTI 常见四个二分维度（强调“偏好”，不等同于能力高低）：

- E–I：能量取向与社交偏好（外向/内向）
- S–N：信息获取偏好（具体经验/抽象可能）
- T–F：决策偏好（逻辑准则/价值与人际）
- J–P：生活方式偏好（结构计划/灵活开放）

## 常见计分逻辑（它是怎么“算出类型”的）

- 多为自陈式问卷：对陈述句按 5～7 点量表选择“像我/不像我”，分别在每个维度上累计分数。
- 维度本质是连续谱，但结果通常用“二分 + 标签”呈现，便于传播和自我叙事。
- 不同网站题库差异很大，因此结果可比性有限。

## 为什么会火

- 低成本获得“身份标签”，适合社交媒体自我介绍、群体归属与话题互动。
- 描述文案常带有“巴纳姆效应”：温和、可套用、偏正向，容易让人觉得“很像我”。
- 用类型讲故事（恋爱、职场、社交）比讲统计分数更易理解，也更易二创传播。

## 常见争议与注意事项

- 信度：不少人隔一段时间重测会变类型，尤其在临界分数附近；情绪与处境影响很大。
- 二分法简化：把连续特质硬切两类，容易造成刻板印象与“贴标签用人”。
- 预测力有限：对绩效、适配、恋爱匹配的强预测说法通常缺乏扎实证据，更多适合作为自我反思线索。
- 版本混用：把 16Personalities 当作 MBTI 的“同一体系”会造成误解。

更稳妥的用法：

- 把结果当“偏好画像/自我探索线索”，不要当诊断或能力证明。
- 关注“维度强度”而非只看四个字母。
- 在重要评估场景优先选择有信效度报告的量表体系（例如大五相关工具）。
