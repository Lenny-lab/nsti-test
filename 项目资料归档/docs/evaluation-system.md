# NSTI 评价体系

NSTI 会先看 4 组偏好，再生成一张更贴近南师语境的结果卡

## 四组偏好

- 能量补给：你更容易在人群与现场中被点燃，还是更习惯在安静空间里恢复
- 观察视角：你更看重具体事实，还是更容易从意象、联想和氛围进入问题
- 逻辑基准：你更偏向规则与结构，还是更在意关系、感受与处境
- 生活步调：你更依赖计划与节点，还是更依赖弹性、留白和临场调整

## 结果输出

- 一个更贴近南师语境的类型名
- 一个校园锚点
- 四组偏好的当前倾向
- 适配场景、协作建议、学习策略和压力提醒

## 使用边界

- NSTI 不是心理诊断工具
- 不用于招生、招聘、评优、选拔或任何筛选用途
- 不建议只根据结果对自己或他人作重大判断
