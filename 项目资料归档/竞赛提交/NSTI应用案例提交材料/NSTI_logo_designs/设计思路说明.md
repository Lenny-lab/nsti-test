# NSTI Logo 设计思路说明

本文件夹收录了 4 组使用绘图模型生成的 NSTI logo 概念图，均为 PNG 格式、纯白色背景。整体方向围绕“南京师范大学校园生活”“偏好测评”“路径与指示器”“四组维度”展开，主色使用深绿、青绿和少量金色，尽量与 NSTI 当前网页的绿色视觉体系保持一致。

## 01_monogram_compass_leaf.png

定位：徽章式主 logo。

设计要点：

- 将 NSTI 字母、指南针、叶片和路径感结合在一起。
- 中央的 S 形动线可以理解为校园路径，也呼应 Student / Type Indicator 中的“探索”和“指示”含义。
- 下方叶片让整体更贴近校园、成长、学习的气质。

适用场景：

- 项目封面
- 竞赛材料首页
- 网站首屏视觉标识
- 圆形头像或社交平台展示

## 02_campus_gate_horizontal.png

定位：横版品牌标识。

设计要点：

- 左侧图形使用校园门廊、路径和节点构成，右侧为 NSTI 字标。
- 门廊暗示大学场景，路径暗示测评过程，几个圆点对应不同偏好维度。
- 横版结构更适合网站导航栏、页眉、PPT 标题页和正式文档。

适用场景：

- 网站 header
- 文档页眉
- PPT 封面
- GitHub README 顶部品牌图

## 03_four_dimension_app_icon.png

定位：应用图标 / favicon 方向。

设计要点：

- 圆形图标中划分出四个区域，分别隐喻 NSTI 的四组偏好维度。
- 图书、校园建筑、社交、叶片等元素对应学习、空间、协作和生活状态。
- 金色指针位于底部，强化“Indicator”的指示器含义。

适用场景：

- 网站 favicon
- App 图标
- 结果卡角标
- 小尺寸展示

## 04_editorial_wordmark.png

定位：编辑感字标 / 报告封面标识。

设计要点：

- 以 NSTI 字母为主体，路径线穿过字母，形成“校园路线”和“偏好轨迹”的联想。
- 小圆点代表测评节点，金色箭头代表最终指向。
- 视觉更克制，适合正式说明文档和海报标题。

适用场景：

- LaTeX/PDF 封面
- 项目介绍海报
- 正式汇报页
- 低装饰、高识别度的场景

## 推荐使用方案

如果只选一个主 logo，建议优先使用 `02_campus_gate_horizontal.png`。它的字标最清楚，横版结构也最适合网页和说明材料。

如果需要一个图标版本，建议配合使用 `03_four_dimension_app_icon.png`，它在小尺寸下更容易作为 favicon 或头像识别。

如果需要做封面或项目视觉主图，可以使用 `01_monogram_compass_leaf.png`，它更有完整徽章感。

如果需要让材料显得更正式、克制，可以使用 `04_editorial_wordmark.png`。

## 后续可定稿方向

绘图模型适合快速探索风格，但最终正式 logo 如果要长期使用，建议再做一轮人工矢量化处理：

1. 保留绿色、青绿、金色三色体系。
2. 统一 NSTI 字母字形，避免不同场景中字标不一致。
3. 将图形标和字标拆分为“横版、竖版、纯图标”三套规范。
4. 输出 SVG、透明背景 PNG、白底 PNG 三种格式，便于网页、PDF 和 PPT 使用。
